Project Description
Overview

This project serves as a practical exercise for learning Git and GitHub fundamentals.
It demonstrates real-world version control scenarios including feature branching,
merge workflows, and remote repository management.
Features

    Version Control: Complete Git workflow with multiple branches and commits

    Remote Management: GitHub integration for collaborative development

    Calculator Module: Python-based calculator with multiple arithmetic functions

    Error Handling: Graceful handling of edge cases like division by zero

Git Commands Practiced

    git init - Initialize repository

    git branch - Create and manage branches

    git checkout - Switch between branches

    git merge - Merge feature branches

    git remote - Connect to remote repository

    git push - Upload changes to GitHub

    git add/commit - Stage and commit changes

Calculator Functions

    Addition

    Subtraction

    Multiplication

    Division (with zero checking)

    Power/Exponentiation

    Modulo/Remainder
